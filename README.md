# Plumbing Website Template (SEO + Lead Generation)

This is a simple, responsive website template designed for small service businesses such as plumbing, electrical, or home repair services.

The main goal of this project is to demonstrate how to build a clean, conversion-focused website with basic on-page SEO implementation.

---

## 🔗 Live Demo

https://mackyfamous.github.io/PlumbingWeb/

---

## 🚀 Features

* Mobile-friendly responsive layout
* Clear call-to-action buttons (Call Now, Get a Quote)
* Click-to-call and WhatsApp integration
* Simple and clean UI focused on conversions
* Google Maps embed
* SEO-friendly HTML structure

---

## 🔍 SEO Implementation

* Proper heading structure (H1, H2, H3)
* Meta title and meta description
* Image alt text support
* Clean and semantic HTML
* robots.txt and sitemap.xml included
* Canonical URL setup

---

## 📄 Page Structure

* Home (Hero + Services + CTA)
* About section (trust building)
* Services section (keyword-focused)
* Contact section (phone, email, WhatsApp)
* Location (Google Maps)

---

## 🛠️ Tech Stack

* HTML5
* CSS (inline for simplicity)
* No frameworks (lightweight and fast)

---

## 📌 Notes

* This is a static website template (no backend)
* Contact form uses basic links (can be upgraded to Formspree or similar services)
* Designed for easy customization and deployment via GitHub Pages

---

## 📈 Use Case

This template is ideal for:

* Local service businesses
* Freelance web developer portfolios
* SEO-focused landing pages
